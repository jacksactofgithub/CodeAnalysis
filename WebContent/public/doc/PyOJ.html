统一说明 ---------- 通过url访问，返回json数据，并支持JSONP，参数都叫`callback` 如`/some/path/?callback=custom`，返回`custom(...)`即可。 接口1. 批量查询学生做题 ------------------------- 请求类型：POST 参数 | 类型 | 说明 ------------|------------|--------- emails | Array | callback | String | 可选 返回值 callback({ 'email1': [ { proNo: '1-1', score: 100, pass: true, trys: 3 } ], 'email2': [] }) 说明：只需要每个人做过的题目。 接口2. 查询所有题目列表 ------------------------- 请求类型：GET 参数：无。（callback同上） 返回值 callback([ { proNo: '1-1', proName: 'Sum', proLevel: '简单|1-10' } ]) 接口3. 查询题目详细说明 ------------------------- 请求类型：GET 参数 | 类型 | 说明 ------------|------------|--------- proNo | String | callback | String | 可选 返回值 callback({ proNo: '1-1', proName: 'Sum', proLevel: '简单|1-10', detail: 'html format' })
